## Leetcode答案

### 说明
leetcode是练习算法能力修炼编程内功非常好的平台，但是官网上解题报告不全，网上的答案也有对有错，为了提升大家刷题的效率，现将每个题目的各种语言的答案(通过测试)总结发布出来，供大家参考。   

每个题目包含c、c++、java、 python、 python3、 javascript、 c#、 go 8种常见语言的答案。  

### 小程序

github功能有限，我们开发了算法学习的小程序，方便大家学习交流， 目前功能有  

1. 每个题目详细解题报告
2. 每个题目的top5答案 
3. 题目收藏和标记功能
4. 题目按照标签和公司分类
